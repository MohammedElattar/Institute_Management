API requests are carried out in three simple steps as given below:
 1) Construction of request body JSON according to the consuming API service.
2) Generate signature using the secret key. (Provided by Mintroute)
3) Sending request body JSON with required Authentication Headers



Signature generation is carried out in following 3 simple steps as given below:

. Convert “string to sign” to HMAC-SHA256 using SECRET KEY
2. Convert the generated HMAC hash into Raw Binary format.
3. Encode the Binary string with Base 64 encoding.
